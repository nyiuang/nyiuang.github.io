Set

定义

Set是一种新的数据结构，类似于数组，但是他的成员的值都是唯一的，也就是没有重复的值。

用法

  1. Set接收一个数组（或类数组对象）作为参数。

  2. Set初始化时会自动去重。

  3. 可以利用Set的特性去做去重。

  4. Set的实例有add方法，表示向Set中添加新成员，如果成员值已存在，则不添加。

    注意：Set加入的值不会进行类型转换再做比较，比较方式类似于“===”，区别是它判断NaN等于自身。

    两个对象总是不想等。

实例的属性和方法

属性

  • Set.prototype.constructor: 构造函数，默认是Set函数
  • Set.prototype.size: 返回Set实例的成员总数

方法

  1. 操作方法

    • add(value) : 添加值，返回Set实例本身。
    • delete(value) : 删除值, 返回Boolean值，表示删除是否成功。
    • has(value) : 返回Boolean值，表示该值是否是Set实例的成员。
    • clear() : 清除所有值，没有返回值。

  2. 遍历方法

    • keys() : 返回键名的遍历器。
    • values() : 返回键值的遍历器。
    • entries() : 返回键值对的遍历器。
    • forEach() : 使用回调函数遍历每个成员。回调函数的参数可以接收三个参数，分别是键值，键名和集合本身。forEach方法可以接收第二个参数，表示绑定的this对象。

  3. 注意

    1. Set的遍历顺序与加入Set的顺序一致。
    2. 因为Set只有键值没有键名，所以keys与values的方法行为一致，都是返回成员值。
    3. 可以直接使用for…of遍历Set。

  4. 应用

    可以实现集合的交并补差集。

WeakSet

定义

WeakSet与Set结构类似，也是不重复值的集合。

WeakSet的成员只能是对象。

WeakSet中的对象都是弱引用，即垃圾回收机制不考虑WeakSet对该对象的引用。

WeakSet不可遍历，没有size属性。

WeakSet有三个方法：add(), delete(), has()。
